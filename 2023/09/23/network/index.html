Protocols process based on time: AFDX → TTE/TTP → TSN

▲ AFDX (ARINC664)
· Composition: Avionics Integrated Computer Subsystem, AFDX End System, AFDX Switch, VL (Virtual Link)
· The difference between AFDX (Avionics Full Duplex Switched Ethernet) and regular Ethernet is that AFDX is a defined network
· Keywords in MAC: ES (End System), QoS (Quality of Service), partitioning, BAG (Bandwidth Allocation Gap), Jitter, Locally Administered, RM (Redundancy Management)
· Keywords in IP: SAP (Service Access Point), TFTP (Trivial File Transfer Protocol), Sampling, Queuing, IP Fragment, Round-robin, CRC (Cyclic Redundancy Check), Peer-to-peer, UDP
· Five function blocks: Filtering & Control, Switching, ES, Monitoring, Configuration Table

RDMA (Remote Direct Memory Access)

▲ Synopsis
A high-performance network communication technology with the advantages of high bandwidth, low latency, no CPU consumption and zero copy. Compared with traditional communication means such as kernel TCP and DPDK, RDMA has obvious advantages in terms of latency, throughput and CPU consumption

▲ Communications Semantics
· SEDN / RECV
· READ
· WRITE

▲ Connection Classification
· RC (Reliable Connection)
· UC (Unreliable Connection)
· UD (Unreliable Datagram)
· RD (Reliable Datagram)
