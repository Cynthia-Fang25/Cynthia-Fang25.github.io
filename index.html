Wireless Communications & Networks (Click to learn more...)

⭐ Only for self-study and revision summaries

Protocols process based on time: AFDX → TTE/TTP → TSN

▲ AFDX (ARINC664)
· Composition: Avionics Integrated Computer Subsystem, AFDX End System, AFDX Switch, VL (Virtual Link)
· The difference between AFDX (Avionics Full Duplex Switched Ethernet) and regular Ethernet is that AFDX is a defined network
· Keywords in MAC: ES (End System), QoS (Quality of Service), partitioning, BAG (Bandwidth Allocation Gap), Jitter, Locally Administered, RM (Redundancy Management)
· Keywords in IP: SAP (Service Access Point), TFTP (Trivial File Transfer Protocol), Sampling, Queuing, IP Fragment, Round-robin, CRC (Cyclic Redundancy Check), Peer-to-peer, UDP
· Five function blocks: Filtering & Control, Switching, ES, Monitoring, Configuration Table

▲ TTP (Time Triggered Protocol)

▲ TSN (Time Sensitive Networking)

Basics

▲ Computing in Memory (CIM)
· Computing Near Memory: The omputational operations are performed by a separate computing chip or module located outside the storage area.
· Computing In Memory: The computational operations are performed by separate computational units located inside the storage area, and the storage and computation may be analogue or digital.

Computing Near Memory is like
‘ The shortest distance between two points is a straight line’,
students need to find a straight and also the fastest way,
to get from the dormitory to the classroom.

Computing In Memory is like
